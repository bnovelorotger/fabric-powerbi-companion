# Dictionary

This folder is the governed layer of the public core.

## Purpose

- capture reusable business meaning
- record governed technical metadata
- preserve changelog and validation expectations
- provide stable context to Codex skills

## Starter status

The shipped content is intentionally small and generic:

- one sample semantic area
- one sample fact table
- one sample dimension table
- three sample measures

Replace or extend it in private workspaces with your real domain content.
